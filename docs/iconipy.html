iconipy

Say goodbye to the tedious hassle of graphic programs! Now you can create stunning icons directly from your Python code. With the look and feel defined right in the code, adjustments are a breeze. Plus, the included icon sets can easily be expanded with your own sets based on font files.

Installation

pip install iconipy

Usage

First you initialize an "IconFactory" with an icon set and look-and-feel settings like this:

from iconipy import IconFactory 

create_button_icon = IconFactory(
                        icon_set = 'lucide', 
                        icon_size = 64, 
                        font_size = 38,  
                        font_color = (0, 0, 0, 255), # black solid
                        outline_color = 'dimgrey', 
                        outline_width = 6,
                        background_color = 'silver', 
                        background_radius = 10
)

Then you create your icons by just passing a name to a function call. Iconipy uses a codepoints dictionary to 'translate' names into characters/icons: 

icon_home = create_button_icon.asPil('house')
icon_reload = create_button_icon.asPil('refresh-cw')
icon_files = create_button_icon.asPil('files')
icon_exit_app = create_button_icon.asPil('log-out')

Depending on your GUI toolkit's whims, you can create PIL Image Objects, ByteIO Objects, Byte-Strings, Raw Pixel Lists, TkPhotoImage Objects, QImage Objects, save to file, and more.

You need to preview an icon? Here we go:

create_button_icon.show('house')

The icon sets you can choose from include: lucide, boxicons, lineicons, material_icons_regular, material_icons_round_regular, material_icons_sharp_regular, and material_icons_outlined_regular.

Feeling adventurous? Dump all the icons to your hard drive and explore:

create_button_icon.saveAll(<path to target directory>)

You need a hand? Check what the icon set has to offer:

print(create_button_icon.search('hand'))

Just want a list with all icon names? No problem:

print(create_button_icon.icon_names)

More info

Visit https://github.com/digidigital/iconipy or https://iconipy.digidigital.de for sample code for the most popular GUI toolkits and detailed documentation.

Iconify vs. iconipy

Iconify is totally unrelated to the iconipy project. Iconify is more mature and powerful, but the focus is on SVG files rather than bitmaps. You can explore it further on their website: https://iconify.design

302                    icon_size: _SizeAttributeType = None,
303                    font_size: int = None,
304                    font_color: _ColorAttributeType = None,
305                    outline_width: int = None,
306                    outline_color: _ColorAttributeType = None,
307                    background_color: _ColorAttributeType = None,
308                    background_radius: int = None,
309                    ) -> dict:
310        '''Modify one or more parameters of the IconFactory object and retrieve a dictionary containing 
311        the updated configuration. Typically, distinct IconFactories are created for different icon 
312        styles, but there may be scenarios where reusing an existing object is desirable.
313        
314            icon_size (int, tuple): The size of the icons in pixels. Single int value or (int, int)
315            font_size (int): The size of the font. Default is icon_size
316            font_color (str, tuple): The color of the font. Name, RGBA-Tuple or hex string
317            outline_width (int): The width of the outline. 0 does not draw an outline
318            outline_color (str, tuple): The color of the outline.  Name or RGBA-Tuple or hex string
319            background_color (str, tuple): The background color. Name or RGBA-Tuple or hex string
320            background_radius (int): The radius of the background corners.        
321        '''            
322        if not icon_size == None:
323            if isinstance(icon_size, int):
324                icon_size = icon_size if icon_size>0 else 1
325            elif isinstance(icon_size, tuple): 
326                icon_size = (icon_size[0] if icon_size[0]>0 else 1, icon_size[1] if icon_size[1]>0 else 1)
327            self._drawing_kwargs['icon_size']=icon_size
328        if not font_size ==None:
329            self._drawing_kwargs['font_size']=self._check_font_vs_icon_size(font_size, icon_size)
330        if not font_color == None:
331            self._drawing_kwargs['font_color']=font_color
332        if not outline_width == None:
333            self._drawing_kwargs['icon_outline_width']=outline_width if outline_width>=0 else 0 
334        if not outline_color == None:
335            self._drawing_kwargs['icon_outline_color']=outline_color
336        if not background_color == None:
337            self._drawing_kwargs['icon_background_color']=background_color
338        if not background_radius == None:
339            self._drawing_kwargs['icon_background_radius']=background_radius if background_radius>=0 else 0                                                                     
340        
341        return self._drawing_kwargs

343    def _check_font_vs_icon_size(self, font_size, icon_size):
344        if isinstance(icon_size, int):
345            smallest_side = icon_size
346        elif isinstance(icon_size, tuple) and len(icon_size)==2 and isinstance(icon_size[0], int) and isinstance(icon_size[1], int):
347            smallest_side = icon_size[0] if icon_size[0] <= icon_size[1] else icon_size[1]    
348        else:
349            raise AttributeError('icon_size must be int or tuple (int,int)')     
350         
351        if isinstance(font_size, int) and font_size <= smallest_side:
352            return font_size if font_size >= 0 else 0  
353        else:  
354            return smallest_side if smallest_side > 0 else 1

356    def _read_codepoints(self):
357        codepoints = {}
358        for icon_set in _ICON_SETS.keys():
359            icon_set_codepoints = {}
360            _METADATA_FILE = _ICON_SETS[icon_set]["METADATA_FILE"]
361
362            if icon_set == "lucide":
363                with open(_METADATA_FILE) as json_data:
364                    codepoint_data = json.load(json_data)
365                for key in codepoint_data.keys():
366                    icon_set_codepoints[key] = codepoint_data[key]["encodedCode"][1:]
367
368            elif icon_set == "boxicons":
369                pattern = re.compile(
370                    r'.*\.(?P<codepoint_key>([a-z0-9]*-){1,4}[a-z0-9]*):.*\n.*"\\(?P<codepoint_value>.*)";'
371                )
372
373                with open(_METADATA_FILE, "r") as boxicons_codepoint_file:
374                    raw_content = boxicons_codepoint_file.read()
375
376                    # Find all matches in the file content
377                    matches = pattern.finditer(raw_content)
378
379                    # Populate the dictionary with the matches
380                    for match in matches:
381                        codepoint_key = match.group("codepoint_key")
382                        codepoint_value = match.group("codepoint_value")
383                        icon_set_codepoints[codepoint_key] = codepoint_value
384
385            elif icon_set == "lineicons":
386                with open(_METADATA_FILE) as json_data:
387                    codepoint_data = json.load(json_data)
388                for key, value in codepoint_data.items():
389                    icon_set_codepoints[key] = hex(value)[2:]
390
391            else:
392                # Material icons
393                with open(_METADATA_FILE) as material_icons_codepoint_file:
394                    for line in material_icons_codepoint_file:
395                        codepoint_key, codepoint_value = line.strip().split()
396                        icon_set_codepoints[codepoint_key] = codepoint_value
397
398            codepoints[icon_set] = icon_set_codepoints
399
400        return codepoints

410    def _draw_character(
411        self,
412        character,
413        font_path,
414        font_size=32,
415        font_color="grey",
416        icon_size=32,
417        icon_background_color=None,
418        icon_outline_width=0,
419        icon_background_radius=0,
420        icon_outline_color="dimgrey",
421    ):
422
423        # Create image
424        if isinstance (icon_size, int):
425            width = height = icon_size
426        elif isinstance (icon_size, tuple) and isinstance (icon_size[0], int) and isinstance (icon_size[1], int):
427            width = icon_size[0]
428            height = icon_size[1]
429        else:
430            raise AttributeError ('icon_size must be of type int or tuple (int, int)')
431        
432        # Add a background
433        if icon_background_color or icon_outline_width:
434            image = self._image_round_background(
435                size = (width, height),
436                fill = icon_background_color,
437                outline = icon_outline_color,
438                outline_width = icon_outline_width,
439                outline_radius = icon_background_radius,
440            )
441        else:
442            image = Image.new("RGBA", (width, height), (0, 0, 0, 0))
443
444        if font_size > 0:
445            # Load font
446            font = ImageFont.truetype(font_path, font_size)
447
448            # Draw character
449            draw = ImageDraw.Draw(image)
450
451            x = width // 2
452            y = height // 2
453
454            draw.text((x, y), character, font=font, anchor="mm", fill=font_color)
455        return image

457    def _image_round_background(
458        self,
459        size = (64,64),
460        fill = "silver",
461        outline = "grey",
462        outline_width = 7,
463        outline_radius = 10,
464        factor = 3,
465    ):
466        """Create and return a background image with rounded corners. Set the outline radius to size/2 to achieve a circular background."""
467        width = size[0] 
468        height = size[1]
469        im = Image.new("RGBA", (factor * width, factor * height), (0, 0, 0, 0))
470        draw = ImageDraw.Draw(im, "RGBA")
471        draw.rounded_rectangle(
472            (0, 0, (factor * width)-1, (factor * height)-1),
473            radius=factor * outline_radius,
474            outline=outline,
475            fill=fill,
476            width=factor * outline_width,
477        )
478        im = im.resize((width, height), Image.LANCZOS)
479        return im
